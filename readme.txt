== Description ==

Stylish Notification Popup is a responsive popup plugin for wordpress to show attention grabbing message to your visitors with call to action button and customization options.

<strong> Here are some cool features of Stylish Notification Popup plugin</strong>

*   Responsive design
*   Cool in and out animation effects
*   Call to action button
*   Clean and light weight
*   Easy to use admin panel

<strong>Upgrade to Pro and get the following features</strong>

*   WYSIWYG editor to enter your message in the backend with media upload Button
*   Customization options to change font size, popup color, button color, button hover color
*   You can set popup delay and Show Every X Day(s) option to avoid disturbance to your visitor

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
